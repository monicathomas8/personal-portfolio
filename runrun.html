Hackathon - Run, Rudolph, Run!

A festive browser-based board game built during my first hackathon. Players choose one of Santa's reindeer and race against Rudolph to claim the coveted red nose. The design was originally inspired by a dark, Tim Burton-esque festive aesthetic — spooky trees, snow effects and moody colours!. I later suggested updating the theme to be a play on Chuck Berry's classic "Run, Rudolph, Run!" song, the game features a dynamic board with various obstacles and power-ups that players can use to gain an advantage.
My role in the project was focused on CSS styling.

  • Built CSS animations including snow falling and winner celebrations.
  • Implemented responsive design to ensure the game is playable on various devices.
  • Animated reindeer movements and interactions within the game.
  • Collaborated with the team to ensure the game's visual elements were cohesive and engaging.
  • Contributed to the overall user experience by enhancing the game's interface and visual appeal.
Result: 🥉 3rd Place.
