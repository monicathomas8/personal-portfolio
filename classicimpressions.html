Classic Impressions

Built from My E-commerce Store Base Template

Classic Impressions is a fully functional e-commerce website built for a client selling luxury plaster mouldings, Art Deco home pieces, architectural fixings, and high-end interior décor. Key features of Classic Impressions include:

  • Product catalog with detailed descriptions and high-quality images.
  • Shopping cart functionality for easy purchasing.
  • Secure checkout process integrated with Stripe for payment processing.
  • Customer account creation and management for order tracking and history.
  • Custom notifications and email alerts for order updates and forms.
  • Admin dashboard for managing products, orders, and customer information.
  • Responsive design to ensure a seamless shopping experience across all devices.
  • Integration with PostgreSQL for robust data management and scalability.
  • Deployment on Railway for reliable hosting and easy updates.

The site was built on top of a custom Django e-commerce base template I developed, a reusable foundation designed to speed up client projects while keeping the codebase clean and easy to extend. While I'm unable to share the client codebase, you can view the e-commerce base template this project was built on via the GitHub link below.
